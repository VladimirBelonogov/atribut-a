A- это тег, предназначенный для создания ссылок, у него есть обязательный атрибут href - этот атрибут указывает на адрес документа на который происходит переход

Стоит сразу указать что ссылки бывают абсолютные и относительные. Абсолютные - ссылки, которые ведут на какой-то ресурс внешний (http:/и любой сайт, либо www. и любой сайт.Относительные- это те ссылки которые ссылаются внутри сайта на себя, эта ссылка может вести на страницу внутри сайта, либо ан какой-то раздел сайта.

Ещё один важный атрибут, тега A, это атрибут target - данный атрибут, указывает где именно нужно открыть ссылку

И у него есть два основный значения, это _blank - открыть ссылку в новом окне, и _self - открыть ссылку в данном окне.

Также стоит упомять, что атрибут A имеет стандартное оформление в HTML, поэтому если мы не будем применять обнуляющий стиль css, то к нему применяться следующие правила.

1.) color: (internal value); // Тоесть по стандарту, текст который ведёт по ссылке подсвечивается обычно синим цветом.Если конкретно, то вот такой

2.) text-decoration: underline; Тоесть по стандарту, текст который ведёт по ссылке ещё и подчеркнут линией снизу.Вот так

3.) cursor: pointer; Тоесть по стандарту, при наведении на текст или область которая ведёт по ссылке, курсор меняется, снизу глянь 2 примера, как именно он меняется.
